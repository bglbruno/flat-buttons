Get started

CDN

Paste the following code into the <head> section of your site's HTML.
<link href="http://bglbruno.github.io/flat-buttons.com/flat-buttons.min.css" rel="stylesheet">

Default CSS

Copy the file into your project. In the <head> of your html, reference the location to your flat-buttons.min.css.
<link href="path/to/flat-buttons.min.css" rel="stylesheet">

Sizes

Add .btn-lg, .btn-sm, or .btn-xs for additional sizes.

Block level button

Create block level buttons—those that span the full width of a parent— by adding .btn-block.

Disabled state

Setting a state disabled for buttons, using the attribute disabled="disabled" for <button>, and the class .disabled for <a> element.
